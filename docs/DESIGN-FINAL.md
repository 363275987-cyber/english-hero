## 第一部分：设计总纲

### 核心理念

**学习 = 冒险，不是"背完再玩"。** 每一个学习动作都是冒险的一部分，不是前置任务。

### 设计原则

1. **好奇心驱动** — 不直接给答案，让孩子先猜、先发现
2. **看听读写融合** — 不是4个模块，是每个环节的4个维度
3. **3秒吸引力** — 每个页面/环节进来就想看下去
4. **主动 > 被动** — 选、拼、编、造句，全程主动参与
5. **变比强化** — 不可预测的奖励最让人上瘾

### 学习流程总览

```
Phase 1          Phase 2           Phase 3        Phase 4        Phase 5
场景引入(2min) → 单词探索(~20min) → 歌谣挑战(3min) → 对话剧场(5min) → Boss战(5min)
  看+听            看+听+读+写         听+读            听+读+写         综合
```

**总时长**：~25分钟（可拆成多天完成Phase 2）
**每日建议**：4-5个新词（Phase 2分天完成）+ 歌谣/对话（完成时触发）+ 0-12个复习词

## 第二部分：5个Phase详细设计

### Phase 1：场景引入（2分钟）

**目的**：用故事和悬念把孩子拉进学习

**脚本**：
```
🎬 开场：精灵Dodo（🦜）飞入屏幕，背后是一座小镇的全景

🦜 Dodo："嘿！我是Dodo！欢迎来到职业小镇~"
   （语音+动画）

📋 场景描述（滚动文字+插画）：
   "小镇要办一场'职业博览会'，但大魔王把所有的
    职业名片都藏起来了！需要你帮忙找回来！"

🗺️ 藏宝图展开：一张泛黄的地图，16个闪烁的问号

🦜 Dodo："第一个谜题已经亮了！你想不想看看是什么？"

👉 按钮：「🔍 揭开谜底」（不显示"下一课"）
```

**看听读写**：看（小镇插画+藏宝图）+ 听（Dodo语音+BGM）

---

### Phase 2：单词探索（核心环节）

**每个单词经过6步闭环**：

```
Step 1 猜词(看+写) → Step 2 听音(听+读) → Step 3 揭秘(看+听+读)
      → Step 4 语境运用(读+写) → Step 5 我编一个(写)
```

**关键设计（第2轮迭代）**：

#### 5种学习路径（避免重复感）

| 路径 | 猜词方式 | 适合 | 占比 |
|------|---------|------|------|
| A 标准 | 看插画→4选1选中文 | 简单词（cook） | 40% |
| B 挑战 | 看插画→直接拼写英文 | 中等词（pilot） | 20% |
| C 极速 | 听发音→选图片 | 有声音特色的词 | 15% |
| D 故事 | 给一句搞笑描述→猜职业 | 有故事性的词 | 15% |
| E 对比 | 给两个相似词→选区别 | 易混淆词（waiter/waitress） | 10% |

#### Phase 2 内部节奏（心跳曲线）

```
强度
  ▲     Step1      Step3           Step5
  │      猜词→紧张  揭秘→爽感       编写→创造
  │        /\        /\              /\
  │       /  \      /  \            /  \
  │      /    \    /    \          /    \
  │     /      \  /      \        /      \
  │    /        \/        \      /        \
  │   /                   \    /
  │  /  Step2 听音→放松     \  /  Step4 语境→中等
  ▼ /                        \/
    词1    词2    词3    词4    词5
```

快（猜词紧张）→ 慢（听音放松）→ 爽（揭秘惊喜）→ 中（语境运用）→ 深（自己编写）

#### 3个"加油站"（每学4-5个词插入一次）

| 加油站 | 形式 | 目的 |
|--------|------|------|
| 复习卡 | 快速闪现已学单词，选对/选错 | 巩固记忆 |
| 抢答 | 限时10秒，快速问答 | 制造紧张感 |
| 连线 | 左边英文右边中文，连线配对 | 主动回忆 |

---

### Phase 3：歌谣挑战（3分钟）

**衔接**：Phase 2全部完成后，16张卡片飞到中央拼成"演唱会门票"🎫

**两种模式**：

**模式1：跟读模式**
```
🎵 歌词逐行高亮 → 播放原版 → 停顿让孩子跟读 → 下一行
   （录不录音可选，不强制）

What do you do? What do you do?      ← 高亮+播放
I'm a doctor, how about you?         ← 停顿，等孩子跟读
I'm a teacher, I teach at school,    ← 高亮+播放
Pilot flies high, that is so cool!   ← 停顿，等孩子跟读
...
```

**模式2：填空挑战**
```
🎵 播放歌谣（挖空关键词），孩子填入正确单词：

What ___ you do? What ___ you do?
I'm a ___, how about you?
I'm a ___, I teach at ___,
___ flies high, that is so ___!

（给4个选项，每个空独立计分）
```

**评分**：5级（S/A/B/C/D），B级以上获得角色卡奖励

**看听读写**：听（有节奏歌谣）+ 读（跟读）+ 写（填空）

---

### Phase 4：对话剧场（5分钟）

**衔接**：歌谣完成后，舞台灯光闪烁，角色卡掉落

**两段对话角色扮演**（使用现有Module 1 dialogues数据）：

**对话1「认识新朋友」**：
```
🎭 场景：学校门口

  NPC-小明:                  🧑 你(选择):
┌──────────────┐         ┌──────────────┐
│ Hi! What do  │         │              │
│ you do?      │         │  (选一句回答)  │
└──────────────┘         └──────────────┘

选项：
A. "I am a doctor. I work in a hospital."
B. "I am a student. I go to school."
C. "I am a pilot. I fly in the sky."

🔊 点击选项后播放发音 → 自动跟读 → 推进剧情
```

**对话2「梦想职业」**：类似设计，6-8个回合

**对话填空挑战版**：对话中挖空关键词，选择填入

**看听读写**：听（NPC语音）+ 读（角色扮演跟读）+ 写（选句/填空）

---

### Phase 5：Boss战（5分钟）

**衔接**：对话结束后，屏幕变暗，Boss剪影出现

**Boss战题目（Module 1，10道）**：

| # | 题型 | 内容 | 看听读写 |
|---|------|------|---------|
| 1 | 看图选词 | 👨‍🍳插图→选cook | 看+读 |
| 2 | 听音选义 | 听/ˈdɒktə/→选"医生" | 听 |
| 3 | 听音拼写 | 听/paɪlət/→拼写pilot | 听+写 |
| 4 | 看义选词 | "护士"→选nurse | 看+读 |
| 5 | 语境填空 | "My father is a ___"→doctor | 读+写 |
| 6 | 语法选择 | "Where ___ she work?"→does | 读+写 |
| 7 | 看图选词 | 📷插图→选photographer | 看+读 |
| 8 | 反向翻译 | "律师"→选lawyer | 读+写 |
| 9 | 语境填空 | "I want to be a ___"→astronaut | 读+写 |

**战后奖励**：星星 + 宝箱 + 剧情推进（名片收集）

## 第四部分：复习系统

### 间隔重复（武器耐久度机制）

| 操作 | 记忆强度变化 |
|------|------------|
| 完成6步闭环 | 初始80 |
| 复习答对 | +15（上限100） |
| 复习答错 | -30（下限0） |
| 自然衰减 | 每天-5（下限20） |
| 连续3次答对 | 提前标记"已锻造🔨" |

**推送规则**：
- 强度≤60时加入当天复习队列
- 每天最多12个复习词
- 固定间隔：Day 1 / 3 / 7 / 14

**5种复习题型**（随机抽1-2种，保持新鲜）：
1. 速射选择（限时5秒，给英文选中文）
2. 反向翻译（给中文选英文）
3. 拼写挑战（听发音拼单词）
4. 语境填空（例句挖空选词）
5. 发音判定（跟读判定准确度）

**防烦机制**：
- 连续5个答对→可提前结束
- 复习≠重新学，每个词不超过30秒
- 每答对一个就给星星（不等最后）

### 错题本系统

**4类错误**：🔤拼写 / 🔄意思搞混 / 📝语法 / 👂听不懂

- 错题优先级高于正常复习
- 当天错→明天必练→额外间隔推送
- UI概念："错题博物馆"🏛️，消灭错题获馆长盖章

## 第五部分：每日学习计划

### 每日节奏（15-20分钟）

```
每日流程：
1. 复习（0-12词，3-5min）  ← 武器保养
2. 补课（3词，2min）        ← 前置基础
3. 新词学习（4-5词，10min）  ← 核心内容
4. 歌谣/对话/挑战（可选）    ← 应用巩固
5. 每日报告（30秒）         ← 成就回顾
```

### Module 1 四天计划

| 天 | 新词 | 复习 | 补课 | 特殊活动 | 总时长 |
|----|------|------|------|---------|--------|
| Day 1 | cook, doctor, nurse, teacher, pilot, waiter | 0 | 3 | - | 15min |
| Day 2 | singer, photographer, reporter, lawyer, astronaut, tour guide | 6 | 3 | 加油站：复习卡+抢答 | 18min |
| Day 3 | security guard, shop assistant, bus driver, waitress | 12 | 3 | 加油站：连线配对 | 18min |
| Day 4 | 0 | 全部复习 | 3 | Phase 3歌谣+Phase 4对话+Phase 5 Boss | 22min |

### 前置补课

- 100个最关键基础词（30动词+30名词+20形容词+20功能词）
- 每天穿插3个，用轻量3步法（听认+快选+联想，40秒/词）
- 补课词与5A课文挂钩（学职业前先补cook/work/help等）
- 约34天完成第一轮
