# 多邻国英语学习交互全拆解 + New Magic 5A 适配方案

> 拆解目标：把多邻国的所有题型、动画、游戏化机制梳理出来，作为 English Hero 的设计参考

---

## 一、交互题型大全（Exercise Types）

### A. 核心题型（占比约 80%）

#### 1. 🏷️ 选择翻译（Translation - Multiple Choice）
```
上方：一句英文/中文提示
下方：3-4 个选项按钮
```
- **交互**：点选正确翻译
- **多邻国新版**：不用母语翻译了，全英文场景。但我们做儿童产品，**保留中英互译**
- **动画**：选中正确 → 按钮变绿 + 向上弹跳 → 音效叮；错误 → 按钮变红 + 抖动 → 显示正确答案高亮
- **5A 适配**：
  - Module 1: "What does a cook do?" → 选 "厨师做什么？"
  - Module 2: "surf the Internet" → 选 "上网"
  - Module 3: "a balanced diet" → 选 "均衡饮食"

#### 2. ⌨️ 填空翻译（Translation - Type Answer）
```
上方：一句英文/中文提示
下方：输入框 + 虚拟键盘
```
- **交互**：手动输入答案
- **反馈**：实时校验，字母错误时红色下划线标出具体位置
- **动画**：全部正确 → 输入框变绿 + 打勾飞出；有错 → 输入框红框 + 下面显示"正确答案是 XXX"（灰色提示文字）
- **5A 适配**：
  - Module 1: "She works in a hospital." → 输入 "她在医院工作。"
  - Module 5: "I have ridden a horse before." → 输入 "我以前骑过马。"

#### 3. 🧩 拼写构建（Word Bank / Build Sentence）
```
上方：一句提示（图片+英文/中文）
下方：一堆打乱的单词/字母块
```
- **交互**：点击/拖拽单词块拼成完整句子
- **特色**：
  - 底部单词块可以点击添加到上方答案区，也可以点击答案区的词移除
  - 每个单词块是独立的"胶囊"形状，带轻微 3D 阴影
  - 空白区有虚线占位符提示需要几个词
- **动画**：
  - 单词块被选中 → 缩小 → 飞到答案区（带弧线轨迹）
  - 放错位置 → 答案区的词弹回底部（弹跳回弹）
  - 完成正确 → 所有词同时变绿 + 从左到右依次弹跳
- **5A 适配**：
  - Module 1: 提示 "他是一名飞行员" → 拼出 "He is a pilot"
  - Module 3: 提示 "你应该少吃快餐" → 拼出 "You should eat less fast food"
  - Module 6: 提示 "我们必须节约用水" → 拼出 "We must save water"

#### 4. 🔊 听音选择（Listening - Multiple Choice）
```
上方：🔊 播放按钮（大圆形，绿色）
下方：3-4 个选项（文字/图片）
```
- **交互**：点播放听发音 → 选出听到的词/句
- **特色**：
  - 可以重复播放（最多 2 次，第二次速度稍慢）
  - 播放时圆形按钮有脉冲波纹动画
  - 语速分级：慢速 / 正常 / 快速
- **动画**：
  - 播放按钮按下 → 缩小再弹回 + 从中心扩散声波圆环（类似水波纹，2-3 圈向外扩散并淡出）
  - 选对 → 播放按钮变绿 ✓
- **5A 适配**：
  - Module 1: 听 "astronaut" → 选 [宇航员图片]
  - Module 4: 听 "spaghetti" → 选 [意大利面图片]
  - 播放 TTS 音频

#### 5. 🗣️ 口语跟读（Speaking - Pronunciation）
```
上方：需要朗读的句子
下方：🎤 麦克风按钮（按住说话）
```
- **交互**：点击麦克风 → 开始录音 → 松开/自动停止 → AI 评分
- **特色**：
  - 录音时麦克风按钮周围有蓝色脉冲圈
  - 评分：1-5 星（或红/黄/绿三级）
  - 显示标注正确/错误的单词（红色标错音，绿色标对音）
  - 跳过按钮（"说不了，跳过"）
- **动画**：
  - 录音中 → 蓝色声波线从左到右滚动（像心电图）
  - 评分结果 → 星星从底部弹出一颗一颗出现（弹跳动画）
- **5A 适配**：
  - Module 1: 跟读 "What does she do? She is a nurse."
  - Module 5: 跟读 "Have you ever tried ice skating?"

#### 6. 🖼️ 图片配对（Image Matching）
```
上方：一张大图 + 英文/中文标签
下方：3-6 个小图网格
```
- **交互**：选出与上方匹配的图片
- **反向变体**：上方是英文词，下方是一排图片，选正确图片
- **动画**：选中 → 图片放大 + 绿色边框；错误 → 图片红框 + 抖动
- **5A 适配**：
  - Module 1: 显示 "chef" → 选 [厨师图片]
  - Module 3: 显示 "vegetables" → 选 [蔬菜图片]

#### 7. 📖 阅读理解（Reading Comprehension）
```
上方：一段短文（3-5 句）
下方：1-3 道理解题（选择/填空）
```
- **交互**：先读文章 → 回答问题
- **特色**：
  - 文章分段展示，每段可以点展开
  - 生词可以点击查看释义（弹出小气泡）
  - 问题类型：选择题、判断题（T/F）、填空题
- **5A 适配**：
  - Module 2: 阅读关于通讯方式的文章 → 回答 "People used to send messages with ___" (quill and ink)
  - Module 4: 阅读食谱 → 回答 "What do you do first?" → 选择正确步骤

#### 8. 🎯 填空补全（Fill in the Blank）
```
上方：一个句子，中间有空白
下方：单词选项（3-4个）或输入框
```
- **交互**：选出/输入正确的词填入空白处
- **特色**：空白处有下划线 + 闪烁光标提示
- **动画**：
  - 选中词 → 缩小飞入空白处 → 空白处弹跳一下
  - 错误 → 词从空白处弹回底部 + 空白处红框闪烁
- **5A 适配**：
  - Module 1: "My mother works in a ___." → 选 hospital / school / factory
  - Module 5: "I have ___ ridden a horse." → 选 never / ever / already

### B. 辅助题型（占比约 20%）

#### 9. 🔗 连线匹配（Match Pairs）
```
屏幕分两列：左边 5 个英文词，右边 5 个中文词
```
- **交互**：先点左边 → 再点右边 → 连线（带弧线）
- **动画**：
  - 连线出现时是虚线 → 实线 + 颜色填充
  - 配对正确 → 线变绿 + 两端圆点闪烁 + 消失
  - 配对错误 → 线变红 + 抖动 + 消失
  - 全部配对完 → 彩色庆祝动画
- **5A 适配**：每个 Module 结尾复习用
  - Module 3: vegetables↔蔬菜, dairy products↔乳制品, snacks↔零食...

#### 10. 📝 单词拼写（Letter Scramble）
```
上方：图片 + 中文含义
下方：打乱的字母
```
- **交互**：把打乱的字母拼成正确单词
- **特色**：
  - 字母块排列在底部（类似 Scrabble 字母牌）
  - 已用的字母变灰，点答案区可以取回
  - 难度分级：给全部字母 → 给部分字母 → 不给字母
- **动画**：
  - 字母块被选中 → 翻转 180° → 飞入答案区
  - 完成正确 → 所有字母变金边 + 星星特效
- **5A 适配**：
  - Module 1: 提示"律师" → 打乱 [l, a, w, y, e, r] → 拼出 LAWYER
  - Module 4: 提示"寿司" → 打乱 [s, u, s, h, i] → 拼出 SUSHI

#### 11. ✅ 判断对错（True or False）
```
上方：一句陈述
下方：两个大按钮：✅ True / ❌ False
```
- **交互**：快速判断陈述是否正确
- **动画**：
  - True 按钮选对 → 绿色满屏 + 大对勾
  - False 按钮选对 → 叉号飞入 + 显示正确说法
- **5A 适配**：
  - Module 3: "Hamburgers are healthy food." → False
  - Module 6: "We should turn off the lights when we leave." → True

#### 12. 🔄 句型变换（Sentence Transformation）
```
上方：一个句子
下方：需要变换成的形式 + 输入区
```
- **交互**：根据提示改写句子
- **5A 适配**：
  - Module 1: "She is a teacher." → 改为问句 "What does she do?"
  - Module 5: "I went fishing yesterday." → 改为 "Have you ever gone fishing?"

## 二、核心动画与动效

### A. 答题反馈动画

| 场景 | 动画描述 | 时长 | 配合音效 |
|------|----------|------|----------|
| ✅ 答对 | 正确选项/区域 → 绿色渐变填充（从下到上）+ 轻微向上弹跳 | 0.5s | 清脆叮声（升调） |
| ❌ 答错 | 错误选项 → 红色闪烁 + 左右抖动（2次，幅度递减） | 0.8s | 低沉嗡声（降调） |
| 💡 纠错 | 显示正确答案 → 从底部滑入 + 金色高亮脉冲 | 0.6s | 提示音 |
| ⏱️ 等太久 | 提示文字出现 → 从透明渐入 + 上下微微浮动 | 0.4s | 轻轻滴声 |
| 🎉 全对通关 | 屏幕中央 → 五彩纸屑 + 大号绿色进度条填满 + Duo 角色弹出庆祝 | 1.5s | 胜利号角 |

### B. 连击动画（Combo/Streak）

```
连续答对时的序列：
第1题 → 正常绿色反馈
第2题 → 绿色稍亮 + 右上角出现 "2x" 数字（弹入）
第3题 → 绿色更亮 + "3x" + 屏幕边缘出现火焰光晕
第5题 → "5x" + 火焰更大 + Duo 戴墨镜出现
第10题 → "10x" + 全屏火焰特效 + "ON FIRE!" 文字弹出
```

- **连击数字**：每次增加时，数字从 1.5x 缩放到 1x + 弹跳
- **火焰特效**：从屏幕底部向上滚动的橙色/红色渐变粒子
- **连击中断**：所有特效消失 + 轻微的"失落"音效

### C. 角色动画（Duo 等吉祥物）

多邻国的角色贯穿整个学习过程，有不同的状态动画：

| 状态 | 角色表现 | 触发条件 |
|------|----------|----------|
| 😊 开心 | Duo 微笑 + 轻轻上下弹跳 | 答对时 |
| 🎉 庆祝 | Duo 举手 + 五彩纸屑背景 | 完成一课 |
| 😢 伤心 | Duo 流泪 + 下垂 | 连续答错 |
| 😤 生气 | Duo 眉毛下压 + 红色背景 | 连击断了 |
| 😴 困倦 | Duo 打哈欠 + 闭眼 | 好几天没学 |
| 💪 鼓励 | Duo 挥拳 + "You can do it!" 气泡 | 复习错题 |
| 📚 学习 | Duo 戴眼镜看书 | 语法讲解 |

**实现方案**：用 CSS sprite sheet + keyframe animation，每个状态是一组 2-4 帧的循环动画

### D. 进度动画

| 元素 | 动画 |
|------|------|
| 经验条（XP Bar） | 获得XP时 → 条从左到右填充 + 数字递增动画（0 → +10）|
| 课程进度圈 | 每完成一题 → 进度圈顺时针填充一格 + 咔嗒声 |
| 单元路径图 | 完成一课 → 节点从灰色变亮 + 星星飞出 + 下一个节点脉冲发光 |
| 升级动画 | 等级数字 → 从旧数字缩小消失 → 新数字放大弹出 + 光环扩散 |

### E. 页面转场

| 转场 | 动画 |
|------|------|
| 题目切换 | 当前题 → 向左滑出 + 渐隐；新题 → 从右滑入 + 渐入（0.3s）|
| 进入课程 | 路径图节点 → 放大居中 + 背景模糊化 + 课桌面板从底部滑入 |
| 退出课程 | 成绩面板 → 从底部弹入 + 粒子庆祝 → "Continue" 按钮脉冲发光 |
| Tab 切换 | 底部 Tab 选中项 → 圆点指示器滑动过渡（弹簧曲线）|

## 三、游戏化元素

### A. 经验值（XP）系统
```
完成一课：+10 XP
速度奖励：+5 XP（全对且无提示）
连击奖励：连击数 × 1 XP
每日目标：默认 20 XP / 可调 30/50/100
```

### B. 连击（Streak）
```
连续每天学习 = Streak +1
断一天 = Streak 归零
保护道具（Streak Freeze）：免罚一次
Streak 达到里程碑：7/30/100/365 天 → 专属动画 + 徽章
```

### C. 生命值（Hearts）
```
初始：5 颗心
答错一题：-1 心
心用完：必须复习旧内容（限时15分钟）或等待恢复（1心/30分钟）
复习正确：恢复 1 心
```

### D. 排行榜
```
周榜：每周一重置，按 XP 排名
分组：10人一组，前3名升级到更高段位
段位：青铜 → 白银 → 黄金 → 铂金 → 钻石
掉段：连续不达标会掉段
```

### E. 成就系统
```
里程碑类："连续7天"、"完成100课"
技能类："拼写大师"、"听力达人"、"口语流利"
挑战类："连续20题全对"、"零失误完成一个单元"
特殊类："午夜学习"、"早起鸟儿"
```

## 四、学习路径设计

### A. 结构层级
```
Section（学段）
  └── Unit（单元）= 1 个主题
        └── Lesson（课程）= 1 次学习（3-5 分钟）
              └── Exercise（练习）= 1 道题
```

### B. New Magic 5A → 多邻国路径映射

```
Section 1: Jobs（职业）
├── Unit 1-1: 职业名称（16个词）
│   ├── Lesson 1: 认识职业（图片配对 × 6）
│   ├── Lesson 2: 职业拼写（字母拼词 × 5）
│   ├── Lesson 3: "What do/does... do?"（句型构建 × 5）
│   └── Lesson 4: 职业小测验（混合题型 × 8）
├── Unit 1-2: 工作地点
│   ├── Lesson 1: "Where do/does... work?"（填空 + 选择）
│   └── Lesson 2: 场景对话（阅读理解）
└── Unit 1-3: 为什么喜欢这份工作
    ├── Lesson 1: "Because..."（句型构建）
    └── Lesson 2: 单元综合（混合 + 口语）

Section 2: E-age（电子时代）
├── Unit 2-1: 通讯工具（15个词）
├── Unit 2-2: to + 动词（不定式）
├── Unit 2-3: 频率表达（How often...）
└── Unit 2-4: 综合复习

Section 3: Healthy Eating（健康饮食）
├── Unit 3-1: 食物分类（20个词）
├── Unit 3-2: more/less/fewer（比较）
├── Unit 3-3: too much/enough（数量）
└── Unit 3-4: 食物金字塔（阅读理解）

Section 4: Food from Countries（国际美食）
├── Unit 4-1: 各国美食（20个词）
├── Unit 4-2: need... to...（需要...来...）
├── Unit 4-3: First/Next/Then/Finally（顺序词）
└── Unit 4-4: 食谱阅读（阅读理解）

Section 5: The Magic of Nature（户外活动）
├── Unit 5-1: 户外活动（18个词）
├── Unit 5-2: too + 形容词
├── Unit 5-3: 现在完成时 Have you...?
└── Unit 5-4: 单元综合 + 口语

Section 6: Save the Earth（保护地球）
├── Unit 6-1: 环保词汇（15个词）
├── Unit 6-2: already/yet + 现在完成时
├── Unit 6-3: should/shouldn't + 建议
└── Unit 6-4: 总复习大挑战
```

### C. 每课内部流程
```
开始（1题新词预览）
  ↓
练习轮（5-8 题，混合题型）
  ├── 新词首次出现 → 显示图片 + 发音 + 中文
  ├── 旧词复习 → 直接出题
  └── 错题 → 同一题型再出一次
  ↓
结束（1题复习重点）
  ↓
成绩面板（XP + 正确率 + 连击）
```

## 五、与 English Hero 现有设计的融合

|-----------|---------|------|

| 角色动画 | ✅ 采纳 | 换成 English Hero 的 IP 角色 |

### English Hero 改造后的交互流程

```
冒险地图（学习路径）
  │
  ├── 普通关卡（每课 5-8 题）
  │   ├── 新词 → 🖼️ 图片配对 / 🔊 听音选择
  │   ├── 句型 → 🧩 拼写构建 / 🎯 填空补全
  │   ├── 复习 → 🏷️ 选择翻译 / 🔄 连线匹配
  │   ├── 阅读 → 📖 阅读理解
  │   └── 口语 → 🗣️ 跟读（可选）
  │
  ├── 精英关（单元小测，8-12题混合）
  │
  └── Boss 关（Section 通关，打 Boss + 武器掉落）
      ├── Boss 出场动画（塞尔达 Boss 登场风格）
      ├── Boss 战 = 限时答题（答对 = 攻击，答错 = 被攻击）
      ├── 武器系统 = 单词武器 + 耐久度
      └── 胜利 → 武器掉落 + XP + 开启下一段
```

## 六、技术实现要点

### Vue 3 组件结构
```
src/components/exercises/
├── TranslationChoice.vue      # 选择翻译
├── TranslationType.vue        # 填空翻译
├── WordBank.vue               # 拼写构建（核心组件）
```

### 动画技术栈
- **CSS Animations + Transitions**：所有微交互（按钮、颜色变化）
- **Vue Transition**：题目切换 `<TransitionGroup>` + `<Transition>`
- **Canvas / CSS Particles**：连击火焰、五彩纸屑、Boss 特效
- **Lottie**（可选）：复杂角色动画（Duo 级别的复杂动画）
- **Web Speech API**：TTS 播放 + 语音识别（口语跟读）
- **Howler.js**（可选）：音效管理
