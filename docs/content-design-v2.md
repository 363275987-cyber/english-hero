# 第2轮：上瘾机制设计

## 一、Phase 之间的衔接设计——让孩子"滑"下去不停

### 设计原则：**「滑梯效应」——每一站都不想下来**

Phase 之间的过渡不是"点击下一页"，而是**一个有悬念的小事件**，让孩子"不得不"点进去看看。

### 具体衔接设计

#### Phase 1 → Phase 2：「神秘邀请函」

| 元素 | 设计 |
|------|------|
| **触发时机** | Phase 1 结尾，精灵说"我发现了一张藏宝图！上面有好多职业的秘密" |
| **视觉** | 一张泛黄的羊皮纸地图缓缓展开，上面标注着 16 个闪烁的问号❓ |
| **钩子** | "第一个谜题已经亮了！你想不想看看是什么？" |
| **按钮** | 不显示"下一课"，显示「🔍 揭开谜底」 |
| **心理学** | 好奇心驱动（好奇心缺口理论：人对"未完成的信息"有强烈求知欲） |

#### Phase 2 → Phase 3：「收集完成庆祝」

| 元素 | 设计 |
|------|------|
| **触发时机** | 16 个单词全部学完（6步闭环走完） |
| **视觉** | 16 张职业卡片飞到屏幕中央，组成一个大拼图 |
| **动画** | 拼图拼完后变成一张"演唱会门票"🎫 |
| **钩子** | "太棒了！你的职业词汇已经全部解锁！现在来挑战一首超酷的职业之歌吧！" |
| **按钮** | 「🎤 登上舞台」 |
| **心理学** | 成就感驱动（刚完成一个大任务，自信心最高，趁热打铁） |

#### Phase 3 → Phase 4：「角色卡掉落」

| 元素 | 设计 |
|------|------|
| **触发时机** | 歌谣挑战完成，获得高评分（至少 B 级） |
| **视觉** | 歌谣唱完后，舞台灯光闪烁，两张角色卡从天而降 |
| **钩子** | "你有新角色卡！选择你的角色，进入对话剧场吧！" |
| **按钮** | 「🎭 挑选角色」 |
| **意外设计** | 如果只拿到 C 级，精灵会说"差一点点！再唱一次就能拿到角色卡哦！"→ 鼓励重玩 |
| **心理学** | 收集欲驱动（角色卡是稀缺奖励）+ 损失厌恶（不想重来，所以认真唱） |

#### Phase 4 → Phase 5：「Boss 预告」

| 元素 | 设计 |
|------|------|
| **触发时机** | 对话剧场演出完毕 |
| **视觉** | 屏幕突然变暗，远处出现一个巨大的黑色剪影（Boss），两个红眼睛亮起 |
| **钩子** | "⚠️ 职业迷宫的 Boss 出现了！它要挑战你掌握的所有职业词汇！你准备好了吗？" |
| **按钮** | 「⚔️ 迎战 Boss」（按钮有震动反馈） |
| **心理学** | 紧张感 + 自信心（刚通过对话剧场，觉得自己准备好了） |

#### Phase 5 结束：「战后奖励」

| 元素 | 设计 |
|------|------|
| **触发时机** | Boss 战胜利 |
| **视觉** | Boss 爆炸成无数星星，金币雨落下 |
| **奖励** | 一次性奖励：经验值 + 星星 + 随机道具（头像框/称号/徽章） |
| **钩子** | "你太厉害了！Module 1 职业迷宫已通关！下一个副本：Module 2 电子迷宫，要来挑战吗？" |
| **按钮** | 「🚀 下一关」或「🏠 回主页」 |
| **心理学** | 高峰结束律（蔡加尼克效应：完成感的快感让人想开始下一个任务） |

### 衔接节奏总览

```
Phase 1（2分钟）→ 好奇心入场
     ↓ 神秘邀请函
Phase 2（10分钟）→ 成就感累积（每个单词都是一个小胜利）
     ↓ 收集庆祝 + 门票
Phase 3（3分钟）→ 表现欲释放（唱歌、跟读）
     ↓ 角色卡掉落
Phase 4（4分钟）→ 沉浸感（角色扮演）
     ↓ Boss 预告（紧张！）
Phase 5（3分钟）→ 高潮释放（打 Boss）
     ↓ 胜利庆祝 + 下一关预告
```

**总时长：约 22 分钟**，控制在孩子注意力衰退之前结束。

## 二、Phase 2 内部节奏——6步快慢交替

### 设计原则：**「心跳曲线」——不能一直平淡，也不能一直紧张**

单词学习的 6 步（猜→听→读→记→用→编）按照节奏分为**三种速度**：

| 速度 | 对应步骤 | 时长 | 感受 | 频率 |
|------|---------|------|------|------|
| ⚡ 快速 | 猜词 | 15-20秒 | 紧张！快选！ | 开头 |
| 🎵 中速 | 听音、跟读 | 30-40秒 | 专注听、认真读 | 中间 |
| 💭 慢速 | 记忆、运用、创编 | 40-60秒 | 思考、创造 | 结尾 |

### 一个单词的完整节奏曲线

```
    ⚡猜词        🎵听音      🎵跟读        💭记忆       💭运用        💭创编
    ┌─┐         ┌──┐        ┌──┐        ┌───┐        ┌───┐        ┌───┐
    │█│         │██│        │██│        │███│        │███│        │███│
    │█│  紧张!  │██│  专注  │██│  专注  │███│  思考  │███│  应用  │███│  创造
    │█│         │██│        │██│        │███│        │███│        │███│
    └─┘         └──┘        └──┘        └───┘        └───┘        └───┘
    15s          30s          30s          40s          50s          40s
                                                 总计约 3 分钟/单词
```

### 但不是每个单词都一样！——5种学习路径变体

为了避免16个单词都走完全一样的6步导致孩子厌烦，我们设计了**5种学习路径变体**：

#### 路径 A：标准路径（约 4 个词）
```
猜词 → 听音 → 跟读 → 记忆（看图+funnyTip） → 运用（对话填空） → 创编（造句）
```
- 代表词：cook, teacher, reporter, waitress
- 特点：节奏明快，步骤完整

#### 路径 B：挑战路径（适用于难词，约 2 个词）
```
猜词（给2次提示机会）→ 听音（放慢速度+音节拆分）→ 跟读（分音节跟读）→ 记忆（funnyTip+funSentence）→ 运用（看图拼写+对话填空）→ 创编（编对话）
```
- 代表词：photographer, shop assistant
- 特点：每个环节给更多支持，不让孩子挫败

#### 路径 C：极速路径（适用于超简单词，约 3 个词）
```
猜词（3选1极速版）→ 听音+跟读（合并为2轮）→ 记忆（funnyTip速览）→ 运用（快速问答）
```
- 代表词：nurse, singer, bus driver
- 特点：跳过创编步骤，快节奏刷过去，中间穿插彩蛋

#### 路径 D：故事路径（适用于有趣词，约 3 个词）
```
猜词（场景故事导入）→ 听音（在故事语境中听）→ 记忆（funSentence动画）→ 运用（续写故事）→ 创编（编一个搞笑版本）
```
- 代表词：pilot, lawyer, astronaut
- 特点：以故事为主线，单词是故事的一部分

#### 路径 E：对比路径（适用于有关联的词，约 2 对，4 个词）
```
猜词（两个词一起看图对比）→ 听音（两个词一起听）→ 记忆（对比记忆）→ 运用（区分使用）
```
- 代表词：waiter/waitress, tour guide/security guard
- 特点：利用词之间的关联，帮助孩子建立词汇网络

### 每4个单词插入一个「加油站」

| 位置 | 设计 | 目的 |
|------|------|------|
| 第4个单词后 | 🏪 **精灵小课堂**：复习刚学的4个词的 funnyTip | 趁热复习 |
| 第8个单词后 | 🎮 **快速抢答**：看图快速选词，10秒倒计时 | 紧张刺激，提升活跃度 |
| 第12个单词后 | 🧩 **词汇连线**：把职业和工具/地点连线 | 降速思考，深度加工 |
| 第16个单词后 | 🎉 **收集庆祝** → 进入 Phase 3 | 情绪高潮 |

### 完整节奏感示意

```
词1 cook(标准A)    词2 doctor(标准A)    词3 nurse(极速C)    词4 teacher(标准A)
                        ⬇️ 🏪 精灵小课堂（30秒放松）
词5 pilot(故事D)    词6 waiter(标准A)    词7 singer(极速C)    词8 photographer(挑战B)
                        ⬇️ 🎮 快速抢答（1分钟紧张）
词9 reporter(标准A) 词10 lawyer(故事D)   词11 astronaut(故事D) 词12 tour guide(对比E)
                        ⬇️ 🧩 词汇连线（1分钟思考）
词13 security guard(对比E) 词14 shop assistant(挑战B) 词15 bus driver(极速C) 词16 waitress(对比E)
                        ⬇️ 🎉 收集庆祝 → Phase 3
```

## 三、正向反馈与惊喜系统

### 3.1 常规正向反馈（高频、即时）

| 行为 | 反馈 | 时机 |
|------|------|------|
| 猜词正确 | ✅ 弹出对勾 + 星星飞入 + 音效（叮！） | 即时 |
| 跟读评分 A | 🌟 大星星 + "发音完美！" + 彩色粒子特效 | 即时 |
| 运用题正确 | 💡 灯泡亮起 + "你用对了！" | 即时 |
| 创编完成（任何内容） | 🎨 小画板 + "你的想象力真棒！" | 即时 |
| 连续答对3题 | 🔥 火焰标记 + "三连击！" | 即时 |
| 连续答对5题 | ⚡ 闪电标记 + "太强了！" | 即时 |

### 3.2 阶段性正向反馈（每个Phase结束时）

| 阶段 | 反馈内容 |
|------|---------|
| Phase 1 结束 | 🗺️ "冒险开始！你获得了藏宝图！" + 获得地图道具 |
| Phase 2 每学4个词 | 📊 进度条推进 + "已完成 X/16" |
| Phase 2 全部完成 | 🏆 "16个职业词汇，全部拿下！" + 卡片翻转动画 |
| Phase 3 歌谣完成 | 🎤 评分展示（S/A/B/C）+ 角色卡掉落 |
| Phase 4 对话完成 | 🎭 "演出成功！" + 观众鼓掌动画 |
| Phase 5 Boss 战胜 | 👑 皇冠 + 金币雨 + 经验值大爆发 |

### 3.3 惊喜系统（低频、不可预测）

**惊喜原则：不可预测的正向刺激最让人上瘾（变比强化程序）**

| 触发条件 | 惊喜内容 | 概率 |
|----------|---------|------|
| 连续答对8题 | 🌈 **彩虹特效** + 彩虹猫精灵出现说"你太牛了！" | 约 10%（需要连对8题才触发） |
| 猜词极快（3秒内） | ⚡ **闪电称号** + "手速王者！" + 屏幕短暂震动 | 约 5% |
| 跟读满分 | 🎵 精灵突然唱一首搞笑歌 | 约 20% |
| 创编内容特别有趣（关键词命中） | 🤣 "笑死我了！这个给你特别奖励！" + 双倍星星 | 约 15% |
| 随机出现 | 🎁 **宝箱掉落**：学习途中随机弹出一个小宝箱，打开获得随机奖励 | 每天学习约出现 1-2 次 |
| 首次学完一个难词（如 photographer） | 🏅 "这是今天最难的一个词！你做到了！" + 特殊徽章 | 首次 |

### 3.4 反馈的声音设计

| 类型 | 音效描述 |
|------|---------|
| 正确 | 短促清脆的"叮！"（愉悦感） |
| 连击 | 递进音阶：叮→叮叮→叮叮叮（层层递进） |
| 获得星星 | 金币声 |
| 升级 | 胜利号角 |
| Boss 击败 | 爆炸声 + 胜利音乐 |
| 宝箱 | 开箱声 + 闪光音 |
| 错误 | 柔和的"嘟"（不是刺耳的蜂鸣，避免负面情绪） |

## 四、紧张感设计——适度焦虑让人专注

### 4.1 倒计时系统

| 场景 | 倒计时时长 | 设计 |
|------|-----------|------|
| 猜词选择 | 10 秒 | 进度条从绿色→黄色→红色渐变，最后3秒有心跳声 |
| Boss 战每题 | 15 秒 | Boss 画面在角落缓慢靠近，营造压迫感 |
| 快速抢答（加油站） | 5 秒 | 超级快！答错不可怕，刺激就行 |
| 创编环节 | **无倒计时** | 创意需要时间，不施压 |

### 4.2 连击系统

```
第1题正确 → ×1（普通）
第2题正确 → ×2（连击！）
第3题正确 → ×3（三连击！🔥）
第4题正确 → ×4（太强了！⚡）
第5题正确 → ×5（无敌模式！🌟）→ 触发惊喜
答错 → 连击归零，但精灵安慰说"没关系，再试试！"
```

**心理学**：连击系统的核心是"损失厌恶"——打到×4时孩子会格外专注，因为不想归零。

### 4.3 Boss 战紧张感

| 设计 | 描述 |
|------|------|
| Boss 造型 | 一个由各种职业图标拼成的巨大怪物（"职业混乱兽"） |
| Boss 生命值 | 10 格血❤️❤️❤️❤️❤️❤️❤️❤️❤️❤️，每答对一题扣一格 |
| 玩家生命值 | 3 条命❤️❤️❤️，答错扣一条，归零则重来（但题目随机变） |
| 紧张音乐 | Boss 战有专属紧张 BGM（低沉的鼓点） |
| 答对动画 | Boss 被击中后退一步，发出怪叫 |
| 答错动画 | 屏幕轻微震动，Boss 嘲笑"哈哈哈你不会吧！" |
| 终极一击 | Boss 最后 1 格血时，屏幕提示"终极一击！"，背景变红，声音更大 |

### 4.4 「适度紧张」的边界

| ✅ 适度紧张 | ❌ 过度焦虑 |
|-----------|-----------|
| 猜词10秒倒计时 | 创编环节也倒计时 |
| 连击归零但可以重来 | 连击归零扣星星 |
| Boss 战有3条命 | 只有1条命 |
| 答错有柔和提示 | 答错有刺耳蜂鸣 |
| 快速抢答限时5秒 | 所有环节都限时 |

## 五、每日学习量与节奏安排

### 5.1 每天学多少个单词？

**答案：4-5 个新词 + 复习旧词**

| 数据 | 说明 |
|------|------|
| Module 1 总词数 | 16 个 |
| 每天新词 | 4-5 个 |
| 完成 Module 1 需要 | 约 3-4 天 |
| 每天学习时长 | 15-20 分钟 |

### 5.2 每日学习流程（Day 1-4）

#### Day 1：热身日
```
🕐 总时长：15分钟

1. 📋 每日任务概览（30秒）
   "今天要探索4个职业：cook, doctor, nurse, teacher"

2. 🗺️ Phase 1 场景引入（2分钟）
   精灵介绍"职业迷宫"的故事

3. 📝 Phase 2 单词学习（10分钟）
   - cook（标准A路径）
   - doctor（标准A路径）
   - nurse（极速C路径）
   - teacher（标准A路径）
   - 🏪 加油站：精灵小课堂

4. 🎁 每日结算（1分钟）
   "今天学了4个词，获得 28 ⭐"
   展示：明日预告（pilot, waiter, singer, photographer）
```

#### Day 2：进阶日
```
🕐 总时长：18分钟

1. 📋 每日任务（30秒）
2. 🔁 复习 Day 1 的 4 个词（2分钟，快速听音选义）
3. 📝 Phase 2 续学（12分钟）
   - pilot（故事D路径）
   - waiter（标准A路径）
   - singer（极速C路径）
   - photographer（挑战B路径）← 今天最难
   - 🎮 加油站：快速抢答
4. 📊 学习报告 + 结算（3分钟）
```

#### Day 3：挑战日
```
🕐 总时长：18分钟

1. 📋 每日任务（30秒）
2. 🔁 复习 Day 1-2 的 8 个词（2分钟，拼写挑战）
3. 📝 Phase 2 续学（12分钟）
   - reporter（标准A路径）
   - lawyer（故事D路径）
   - astronaut（故事D路径）
   - tour guide（对比E路径，和 security guard 配对）
   - 🧩 加油站：词汇连线
4. 📊 学习报告 + 结算（3分钟）
```

#### Day 4：冲刺 + Boss 战日
```
🕐 总时长：20分钟

1. 📋 每日任务（30秒）
2. 🔁 复习全部 12 个词（2分钟）
3. 📝 Phase 2 续学（8分钟）
   - security guard（对比E路径）
```

### 模式设计：两种模式

**流程**：
```

   第1次：TTS 领读 → 孩子跟读
   第2次：只有背景音乐 → 孩子自己读（文字显示）
   第3次：TTS 领读 → 孩子跟读（巩固）

3. 🎶 完整跟读（带背景音乐和节拍器）
   所有6行连起来跟读一遍
```

**视觉设计**：
- 每行对应一个职业角色，该角色在做对应动作的动画
- 文字有 KTV 式的高亮滚动效果
- 背景音乐是欢快的鼓点节奏

**无评分**，结束后显示："你学会了！准备好挑战了吗？"

**【按钮】** `🎵 进入挑战模式`

#### 模式B：挑战模式（Challenge 模式）

**目的**：在歌谣中填空，测试记忆

**形式**：歌谣播放，但每行有一个关键词被遮住，孩子需要填入。

**6行歌谣的填空设计**：

| 行号 | 歌谣原文 | 填空词 | 选项 |
|------|---------|--------|------|
| 1 | What do you do? What do you do? | （不填空，热身行） | — |
| 2 | I'm a **doctor**, how about you? | doctor | doctor / teacher / cook |
| 3 | I'm a **teacher**, I teach at school, | teacher | teacher / nurse / singer |
| 4 | **Pilot** flies high, that is so cool! | Pilot | Pilot / Lawyer / Waiter |
| 5 | **Photographer** takes photos all day, | Photographer | Photographer / Reporter / Shop assistant |
| 6 | **Reporter** has news to say! | Reporter | Reporter / Lawyer / Tour guide |

- 每行播放时暂停，等孩子选择

**评分系统**：

## Phase 5：Boss 战题目设计

### Boss 设定

- **Boss 名称**：职业混乱兽（Jobs Chaos Monster）
- **Boss 造型**：由各种职业图标（听诊器、锅铲、飞机翅膀、相机、话筒等）拼成的巨大怪物，红眼睛
- **Boss 台词**："你以为学会了所有职业？来挑战我吧！"
- **Boss 血量**：10 格 ❤️❤️❤️❤️❤️❤️❤️❤️❤️❤️
- **玩家生命**：3 条命 ❤️❤️❤️
- **BGM**：紧张鼓点

### 10道Boss战题目（覆盖6种题型）

> **题型说明**：
> 1. 看图选词（图片 → 单词）
> 2. 听音选义（声音 → 中文意思）
> 3. 听音拼写（声音 → 拼写单词）
> 4. 看义选词（中文意思 → 英文单词）
> 5. 语境填空（对话中的词）
> 6. 语法选择（正确语法形式）

---

#### 题目1：看图选词 🖼️

**【Boss动画】** Boss 扔出一个光球，光球变成图片

**【画面】** 一个穿白大褂的人在给病人检查

**【问题】** What is this person's job?（这个人的职业是什么？）

**【选项】**
```
A. cook       B. doctor       C. teacher
```

**正确答案**：B. doctor

**答对**：Boss 被击中退后一步！"啊！"

#### 题目5：语法选择 📖

**【Boss动画】** Boss 召出一块巨大的黑板

**【问题】** 选择正确的句子：

**【选项】**
```
A. She work in a hospital.
B. She works in a hospital.
C. She working in a hospital.
```

**正确答案**：B. She works in a hospital.（she后面动词加s）

**答对**：Boss 摇晃！"语法也不怕？！" + Boss 血量只剩一半❤️❤️❤️❤️❤️
