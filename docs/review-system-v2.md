## 一、间隔重复系统（Spaced Repetition System）

### 1.1 核心设计理念

**"武器耐久度"机制** — 把每个单词想象成游戏里的武器，不用就会"生锈"，复习就是"保养"。

- 🟢 **满耐久（100%）**：刚学会，记忆清晰
- 🟡 **中耐久（50-80%）**：快忘了，需要复习
- 🔴 **低耐久（<50%）**：快锈死了，急需保养
- ⚫ **已损坏（0%）**：完全忘记，需要重新学习

> 用"保养武器"代替"复习"，孩子不会觉得在"重复劳动"，而是在"维护自己的装备"。

### 1.2 记忆强度（Memory Strength）计算

每个单词维护一个**记忆强度值（0-100）**：

| 状态 | 强度范围 | 说明 |
|------|---------|------|
| 全新 | 0 | 从未学过 |
| 刚学会 | 80 | 完成6步学习闭环后初始值 |
| 复习正确 | min(100, 当前值 + 15) | 复习答对，强化记忆 |
| 复习错误 | max(0, 当前值 - 30) | 复习答错，大幅衰减 |
| 自然衰减 | 每天自动 -5（最低不低于20） | 模拟遗忘曲线 |

**简化原则**：只有5个操作（学完→+80，答对→+15，答错→-30，每天→-5），不需要Anki那样复杂的SM-2算法。

### 1.3 复习推送时机

```
学完 → Day 1 → Day 3 → Day 7 → Day 14 → 长期记忆（不再推送）
         ↑        ↑        ↑         ↑
       第1次    第2次    第3次     第4次
       复习      复习      复习      复习
```

**推送规则**：
- 记忆强度降到 **≤60** 时，自动加入当天的"保养队列"
- 每天最多推送 **12个** 复习单词（避免过载）
- 如果当天复习队列超过12个，按"最紧急"排序（强度最低的优先）
- 如果复习全部通过（4次间隔全部答对），标记为 **"已锻造" 🔨**（长期记忆）

### 1.4 复习题形式（与学习时不同）

学习时用的是"6步闭环"（听→认→跟读→拼→用→趣），复习时必须**换形式**，避免机械重复：

| 复习形式 | 说明 | 游戏化包装 |
|---------|------|-----------|
| **速射选择** | 给出英文，4选1选中文（限时5秒） | "射击训练"——快速消灭怪物 |
| **反向翻译** | 给出中文，选出正确英文 | "密码破译"——破解敌人的密码 |
| **拼写挑战** | 听发音，拼出单词（可给首字母提示） | "锻造武器"——打造自己的装备 |
| **语境填空** | 例句挖空，选词填入 | "任务关卡"——用正确的武器完成任务 |
| **发音判定** | 跟读单词，AI判定发音准确度 | "吟唱法术"——念对咒语获得能量 |

**每次复习随机抽1-2种形式**，保持新鲜感。

### 1.5 复习结果 → 游戏进度

| 复习结果 | 游戏效果 |
|---------|---------|
| 全部答对（100%） | 武器升级 ✨ + 获得双倍星星 + 解锁"锻造大师"进度 |
| 答对≥80% | 武器耐久恢复 + 正常星星奖励 |
| 答对50-80% | 武器耐久部分恢复 + 明天继续复习这个单词 |
| 答对<50% | 武器"受损" ⚠️ + 错题自动进入错题本 + 明天强制复习 |

### 1.6 防烦机制（Anti-Fatigue）

| 策略 | 具体做法 |
|------|---------|
| **复习上限** | 每天最多12个复习词，超出推到明天 |
| **连续正确跳过** | 同一个单词连续3次复习全对，提前标记为"已锻造" |
| **轻松模式** | 如果连续5个都答对，提示"今天的保养任务完成！"可以提前结束复习 |
| **复习 ≠ 重新学** | 复习题快节奏、短时间，每个词不超过30秒 |
| **奖励反馈** | 每完成一个复习词就给星星，不是最后才给 |

## 二、每日学习计划

### 2.1 标准每日节奏（15-20分钟）

```
┌─────────────────────────────────────────────────┐
│  ⏰ 每日学习流程（总计15-20分钟）                  │
├─────────────────────────────────────────────────┤
│                                                 │
│  ① 任务概览（30秒）                              │
│     "今天要锻造3把新武器，保养5把旧武器！"          │
│                                                 │
│  ② 新单词学习（8-10分钟）                        │
│     5-6个新单词 × 6步闭环 ≈ 1.5分钟/词           │
│                                                 │
│  ③ 武器保养/复习（5-8分钟）                      │
│     间隔重复队列（≤12个） + 错题本队列            │
│                                                 │
│  ④ 每日结算（1-2分钟）                           │
│     看星星、看成就、看"武器库"                    │
│                                                 │
└─────────────────────────────────────────────────┘
```

### 2.2 关键参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 每日新词 | 5-6个 | 不超过6个，避免认知过载 |
| 每日复习 | 0-12个 | 根据间隔重复队列动态调整 |
| 每日总时间 | 15-20分钟 | 上限25分钟，超时提醒"今天的训练已完成，明天见！" |
| 每日最少天数 | 0天 | 没有强制打卡，但鼓励连续 |

| 步骤 | 具体内容 |
|------|---------|

| 步骤 | 内容 | 时间 |

Module 1学完后，16个单词自动进入间隔重复队列：

```
Day 4  → Day 1的6个词 第7天复习 (cook, doctor, nurse, teacher, pilot, waiter)
```

同时每天还有Module 2的新词学习（5-6个/天），所以实际每天的复习量会叠加，但系统有**12个上限**保护。

## 三、前置补课计划

### 3.1 设计原则

**❌ 不这么做**：先花2个月补完300个基础词，再开始学5A
**✅ 我们这么做**：每天学5A新词的同时，穿插补3-4个高频基础词，"边走边补"

**为什么？**
- 孩子已经转学了，需要尽快跟上5A进度
- 基础词缺口虽然大，但不是每个都 equally urgent
- 每天补3-4个，100天就能补300个，完全来得及

### 3.3 补课穿插方案

```
每日学习计划（更新版）：
┌──────────────────────────────────────────┐
│  ① 任务概览（30秒）                       │
│  ② 新单词学习（8-10分钟）→ 5A课程         │
│  ③ 即时练习（2分钟）                      │
│  ④ 前置补课（3分钟）→ 3个基础词            │
│  ⑤ 武器保养/复习（5分钟）                 │
│  ⑥ 每日结算（1分钟）                      │
└──────────────────────────────────────────┘
```

**补课策略**：
- 每天学 **3个** 前置基础词，放在新词学习之后
- 补课形式**更轻量**：不需要完整的6步闭环，用3步就够了（听→认→选）
- 100个基础词 ÷ 3个/天 ≈ **34天** 完成第一轮补课
- 补课词汇也纳入间隔重复系统，但频率更低（3天、7天、14天，跳过第1天）

### 3.4 补课的轻量3步法

跟主课程的6步闭环不同，补课词用简化版：

| 步骤 | 内容 | 时间/词 |
|------|------|---------|
| 🎧 听+认 | 播放发音 + 展示趣味图片 + 显示单词 | 15秒 |
| ✅ 快速选择 | 看英文选中文（2选1，降低难度） | 10秒 |
| 💡 趣味联想 | 一句话记忆法（预设或孩子自己编） | 15秒 |
| **合计** | | **40秒/词** |

3个词 × 40秒 = **2分钟**，非常轻松。

### 3.5 补课词汇与5A教材的关联设计

补课词尽量跟当天学的5A词汇**挂钩**：

| 5A当天学 | 补课词选择 | 关联方式 |
|---------|-----------|---------|
| cook, doctor, nurse... | make, work, help | "A **cook** **makes** food. A **doctor** **helps** people." |
| computer, smartphone... | use, read, play | "We **use** a **computer** to **read**." |
| vegetables, rice, fish... | eat, drink, good | "We **eat** **vegetables**. They are **good**." |
| curry, pizza, sushi... | like, want, new | "I **like** **pizza**. I **want** to try **new** food." |
| play badminton, fly kites... | play, go, have | "Let's **play** badminton. Let's **go** fly kites." |
| save water, recycle... | can, should, every | "We **should** **save** water **every** day." |

> 用关联的方式，补课词不再是"额外的负担"，而是理解5A词汇的**工具**。

## 四、错题本系统

### 4.1 错题记录方式

**什么时候记录错题？**
- 复习时答错 → 自动记入错题本
- 模块测试时答错 → 自动记入错题本
- 新词学习时拼写错 → 提示"记入练习本"，但不进错题本（学习阶段允许犯错）

**记录什么？**

```json
{
  "word": "photographer",
  "error_type": "spelling",
  "wrong_answer": "photograper",
  "correct_answer": "photographer",
  "context": "Module 1 - Day 2 复习",
  "timestamp": "2026-03-26",
  "attempts": 1,
  "last_attempt": "2026-03-26"
}
```

### 4.2 错题分类

| 错误类型 | emoji标识 | 具体表现 | 针对练习 |
|---------|----------|---------|---------|
| 🔤 拼写错误 | 🔤 | 认识但写错（如 photographer 漏了 h） | 拆音节拼写训练：pho-to-graph-er |
| 🔄 意思搞混 | 🔄 | cook/chef 分不清，waiter/waitress 搞混 | 对比配对练习：图文连线 |
| 📝 语法错误 | 📝 | 知道词但不会用（如 "She is a nurse" 写成 "She is nurse"） | 例句填空练习 |
| 👂 听不懂 | 👂 | 看到认识，但听不出来 | 听力辨析训练 |

### 4.3 错题推送策略

```
错题复习优先级 > 正常间隔重复复习
```

| 规则 | 说明 |
|------|------|
| **当天再练一次** | 错题在学习结束后立刻推送一次"挑战模式"（答对3题才通过） |
| **明天必练** | 错题自动加入第二天的复习队列（不受12个上限限制） |
| **+1次额外复习** | 在正常间隔重复的基础上，多推一次（比如原来3天后再复习，错题改为1天后+3天后+7天后） |
| **3次连续答对** | 同一个错题连续3次答对，退出错题本 |
| **专属练习** | 根据错误类型，推送针对性的练习（见上表） |

## 七、关键技术参数汇总

| 参数 | 值 | 备注 |
|------|-----|------|
| 新词/天 | 5-6个 | 不超过6个 |
| 补课词/天 | 3个 | 轻量3步法 |
| 复习词/天 | 0-12个 | 上限保护 |
| 每日总时间 | 15-20分钟 | 上限25分钟 |
| 记忆强度初始值 | 80 | 学完后 |
| 复习答对增幅 | +15 | 上限100 |
| 复习答错降幅 | -30 | 下限0 |
| 自然衰减 | -5/天 | 下限20 |
| 复习触发阈值 | ≤60 | 加入复习队列 |
| 间隔复习节点 | Day 1, 3, 7, 14 | 固定间隔 |
| 错题退出条件 | 连续3次答对 | 退出错题本 |
| 锻造完成条件 | 4次间隔全对 | 标记为"已锻造"长期记忆 |
| 补课完成周期 | ~34天 | 100词 ÷ 3个/天 |
| 模块学习周期 | ~3天/模块 | 104词 ÷ 6模块 ≈ 20天学完全部新词 |

> 💡 **星姐的教研笔记**：
> 
> 这个系统的核心是"**轻量但有效**"。间隔重复不能搞得太重——孩子是来玩游戏的，不是来做题的。
> "武器耐久度"这个包装让复习变成了"保养装备"，心理负担完全不同。
> 补课词跟5A课文挂钩，不是"额外负担"而是"理解工具"。
> 错题博物馆把负面体验（做错题）变成了正面体验（收集展品、盖章消灭）。
> 数据报告要有"人情味"，不要冷冰冰的数字——孩子要的是"被看见"，不是"被统计"。
